# Faculty-Publication-Management-System
A web-based system for managing faculty publications (journals, conferences, and books) with role-based access, advanced search filters, and admin approval workflows using Node.js and MySQL.
# Features
Role-based login for faculty and admin
Add, update, and request deletion of publications (admin approval required)
Search and filter publications by faculty, academic year, and research area
Detailed publication views including co-authors and metadata
View all publications categorized as journals, conferences, or books
# Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MySQL
Deployment: Localhost / Any Node.js supported server
